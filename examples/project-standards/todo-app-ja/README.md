# Project Standards Example

このサンプルは `.specify/project/` 配下に置く最小の共通統制文書を示します。

- `.specify/project/tech-stack.md`
- `.specify/project/coding-rules.md`
- `.specify/project/architecture-principles.md`
- `.specify/project/domain-map.md`

feature 固有の UI / flow / test 設計は `designs/specific_design/`、共有 API / Data / Module の正本は `designs/common_design/` に置きます。
