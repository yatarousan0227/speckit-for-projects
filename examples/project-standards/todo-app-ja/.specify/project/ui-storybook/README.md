# Project UI Storybook

このサンプルは、ToDo アプリのローカルデザインシステムに対する project-level の UI 基盤を示す。

feature ごとの設計束を作る前に、Atomic Design の部品と主要画面レイアウトをレビューするために使う。

## Purpose

- `atoms`、`molecules`、`organisms`、`templates`、`pages` を定義する
- feature ごとの設計に入る前に主要画面レイアウトを確認する
- `.general_sdd/project/design-system.md` と Storybook のサンプルを対応させる
