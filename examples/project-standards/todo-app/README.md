# Project Standards Example

This sample shows the minimal repository-wide standards expected under `.specify/project/`.

- `.specify/project/tech-stack.md`
- `.specify/project/coding-rules.md`
- `.specify/project/architecture-principles.md`
- `.specify/project/domain-map.md`

Feature-level UI, flow, and test design belongs under `designs/specific_design/`. Shared API, data, and module truth belongs under `designs/common_design/`.
