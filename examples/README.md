# Example Projects

This directory contains three example outputs that demonstrate the expected `brief -> design -> tasks` deliverables for:

- screen-centric projects
- API-centric projects
- batch-centric projects

Each example includes a brief and a complete design bundle with `traceability.yaml` and `tasks.md`.

It also includes project-standards examples for a ToDo application under:

- `project-standards/todo-app/` for the English sample
- `project-standards/todo-app-ja/` for the Japanese sample
